# Logical Grouping & Organization
- Slashes create hierarchical namespaces that group related branches together
	- **feature/** - New feature
	- **bugfix/** - Bug fixes
	- **hotfix/** - Urgent production fixes
	- **release/** - Release preparation
## Naming Convention
- type/username/description
	- `feature/nate/api-refactor`
