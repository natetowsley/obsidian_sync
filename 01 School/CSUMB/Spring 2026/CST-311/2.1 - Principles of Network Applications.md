## Socket
- Process sends/ receives messages to/from its socket
- Socket analogous to door
	- sending process shoves message out door
	- sending process relies on transport infrastructure on other side of door to deliver message to socket at receiving process
![[Pasted image 20260211140846.png|450]]
- Socket allows transmission of data between two processes on the same machine or different machines over a network
- Commonly used in client-server interaction, as it allows applications to communicate using the build-in mechanisms of the hardware and operating system
	- The client acts as the requester, where it requests some data
	- The server acts as the listener and provides the client the requested data as the response
### Socket Programming
- Socket programming shows how to use socket application program interfaces (APIs) to establish communication links between remote and local processes
- Many APIs are available for most programming languages
